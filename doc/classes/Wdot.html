Node_pat = /^\s*(\w+[-\w]*)\b\s*([^(?:->)\[]*)$/

Start_node_pat = /^\s*(_(\w+[-\w]*))\b\s*([^(?:->)\[]*)$/
  Pattern - start_node definition eg. _begin The parse patern is almost the same - just different capture groups.
